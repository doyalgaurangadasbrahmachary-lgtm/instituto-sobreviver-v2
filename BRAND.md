## 2. Sistema de Diseño (Design Tokens)

### 2.1 Paleta de Color (OKLCH)
La paleta evoca calma, confianza y espiritualidad. Se ha convertido a OKLCH para una mejor fidelidad perceptiva en pantallas modernas.

| Token | Rol | HEX | OKLCH (Aprox) | Uso |
| :--- | :--- | :--- | :--- | :--- |
| `brand-cream` | Fondo / Base | `#f1f1ea` | `oklch(95% 0.02 105)` | Fondos generales, tarjetas suaves |
| `brand-navy` | Primario / Texto | `#24526e` | `oklch(42% 0.09 235)` | Encabezados, textos principales, botones sólidos |
| `brand-cyan` | Acento / Acción | `#34bbce` | `oklch(73% 0.16 235)` | Botones secundarios, iconos, detalles destacados |
| `brand-white` | Superficie | `#ffffff` | `oklch(100% 0 0)` | Contenedores internos, modales |

### 2.2 Tipografía (Sugerida - Google Fonts)
Seleccionada para transmitir profesionalismo premium y calidez humana.

*   **Display / Encabezados:** `Playfair Display` (Serif).
    *   *Razón:* Elegante, clásica, evoca seriedad y respeto. Ideal para "Instituto Sobre'Viver" y citas inspiradoras.
*   **Cuerpo / UI:** `Lato` o `Inter` (Sans-serif).
    *   *Razón:* Legibilidad alta, moderna, limpia. Balancea la seriedad de la serif.

### 2.3 Espaciado y Formas
*   **Radio de Borde:** `Rounded-xl` (aprox. 12px-24px) para evocar suavidad y bienvenida. Evitar esquinas afiladas agresivas.
*   **Espaciado:** Generoso (`p-8`, `gap-6`). Diseño "respirable" para transmitir paz.

## 3. Guía de Estilo UI (Awwwards Standard)
*   **Estética:** "Clean Soul". Minimalismo cálido.
*   **Sombras:** Suaves y difusas (`shadow-lg` con opacidad baja en tonos azules) para dar elevación sin peso visual.
*   **Micro-interacciones:** Transiciones lentas y suaves (`duration-300`, `ease-out`). Nada brusco.
